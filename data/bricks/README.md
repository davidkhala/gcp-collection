## Provision

- Workspace name cannot be `databricks`. It is a reserved
- Personal Compute Cluster create/start time: 25 minutes
- Not available in Hongkong region `asia-east2`, alternative Asia available region are:
  - `asia-northeast1` Tokyo
  - `asia-south1` Mumbai
  - `asia-southeast1` Singapore

- No serverless compute for notebook (Dec 2024)
