
# [pricing](https://cloud.google.com/data-fusion/pricing)

Google Cloud Data Fusion 是按“实例运行小时数 + Spark 执行成本”计费的
- > usage is measured as the length of time, in minutes, between the time a Cloud Data Fusion instance is created to the time it is deleted.
  - There is no way to pause/stop
- > For pipeline execution, you are charged for the Managed Service for Apache Spark(dataproc) clusters that Cloud Data Fusion creates to run your pipelines

Tiers
- Basic
  - first 120 hour is free
  - $1.80 per hour
- Developer
  - $0.35 per hour
- Enterprise
  - $4.20 per hour
